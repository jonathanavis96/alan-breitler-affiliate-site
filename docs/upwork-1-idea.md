## Detailed Description

This is a client project from Upwork for a user named Alan who wants an affiliate marketing website to attract traffic and generate income. The site will be custom-built (not template-driven) with source code handover.

Key characteristics:
- Affiliate-focused, content-driven site
- Hub-and-spoke content architecture (pillar pages linking to reviews, comparisons, guides)
- Internal affiliate link redirect system (/go/[slug]/)
- Product review cards, comparison tables, pros/cons lists
- SEO-first: static HTML output, JSON-LD schema markup, XML sitemap
- Mobile-first responsive design
- Placeholder branding ("Company Name") until client provides details

The niche, audience, and branding are not yet confirmed by the client. Using sensible defaults that can be updated when client responds.
