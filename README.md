# Company Name - Affiliate Website

Custom-built affiliate marketing website designed to attract traffic and generate income through affiliate partnerships.

## Tech Stack

- **Astro** - Static site generation for fast, SEO-optimized pages
- **Tailwind CSS** - Utility-first styling
- **TypeScript** - Type-safe development
- **MDX** - Content authoring with components

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

- `src/` - Source code (components, layouts, pages, styles)
- `content/` - MDX content files (reviews, roundups, guides, blog)
- `data/` - JSON data files (products, affiliate links, site config)
- `public/` - Static assets (images, fonts)
- `docs/` - Project documentation

## Development

Work happens on feature branches. Use PRs back to main for releases.
